# 新手引导 Onboarding Tutorial 设计文档

> **创建时间:** 2026/04/22
> **更新:** 2026/04/23 (新增案例展示步骤)

---

## 1. 概述

**目标:** 首次进入应用的用户看到 3-4 步引导 tutorial，介绍八字报告是什么、如何使用、可以获取什么。降低用户流失率，提升首次使用完成率。

**最终设计:** 5 步引导流程（包含新增的案例展示步骤）

---

## 2. 引导流程

### Step 1: 欢迎页
- 品牌展示：Logo + 标语「AI 八字命理分析 · 探索你的命运密码」
- 底部圆点指示器（5点，当前高亮）
- CTA 按钮：「开始探索 →」

### Step 2: 什么是八字
- 图示：天干（☰）+ 地支（☷）组合示意
- 说明文案：八字 = 出生时的时空密码，由天干和地支组成，蕴含人生轨迹与命运走向
- 底部圆点指示器 + 「下一步 →」按钮

### Step 3: 如何使用
- 3 步流程卡片：
  1. 输入出生信息（姓名、性别、出生年月日时）
  2. AI 智能分析（深度解读事业、感情、财运、健康）
  3. 获取专属报告（详细命盘分析 + 未来大运走势）
- 底部圆点指示器 + 「下一步 →」按钮

### Step 4: 案例展示
- 显示示例报告卡片（模拟真实用户「小明」的命盘）
- 示例数据：
  - 姓名：小明
  - 性别：男
  - 出生年份：1995年
  - 生肖：猪
  - 五行：火
  - 雷达图评分：事业85分、感情72分、财运68分、健康90分
  - 一句话总结：事业心强，财运平稳。2026年有晋升机会，注意肝胆健康...
- 作用：让用户在填写信息前看到预期结果，提升转化
- 底部圆点指示器 + 「下一步 →」按钮

### Step 5: 隐私说明
- 🔒 图标 + 文案：
  - 出生信息仅用于命理计算
  - 数据存储在本地设备
  - 可随时删除所有数据
- 底部圆点指示器 + 「开始使用」按钮（主 CTA）

## 3. 技术实现

### 组件结构
```
src/components/OnboardingTutorial.tsx    # 主组件，管理步骤状态
```

### 状态管理
- `currentStep`: number (0-4)
- `isVisible`: boolean (是否显示引导)
- localStorage key: `onboarding_completed` (boolean)

### 触发条件
1. 首次打开应用时检查 localStorage
2. 如未完成引导，显示 OnboardingTutorial
3. 完成引导后设置 `onboarding_completed = true`
4. 用户可随时在设置中重置引导

### 动画效果
- 页面淡入 + 向上滑动 (opacity 0→1, translateY 20px→0, 300ms ease-out)
- 步骤切换：当前步骤淡出 + 新步骤淡入
- 底部圆点：当前高亮 (金色 #d4af37)，其他半透明白色

### 跳过机制
- 每步右上角「跳过」文字链接
- 点击直接完成引导
