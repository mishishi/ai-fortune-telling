## 功能1：首页每日运势卡片

### 设计方案
- **样式：** 丰富版（B方案）- 综合运势分 + 四维评分 + 命主提示 + 今日宜忌 + 吉时
- **位置：** 首页顶部，Birth Form 上方（Hero位置）
- **数据来源：** 基于用户历史报告数据计算
- **未登录/无报告用户：** 显示引导卡片"生成你的专属命盘"

### 卡片内容
```
┌─────────────────────────────┐
│ 📅 今日运势（甲辰年 三月初三）│
│─────────────────────────────│
│ ☀️ 82分 综合运势优秀         │
│ [💼85] [💕78] [💰72] [🏥88] │
│ 💡 命主提示：今日贵人运旺盛   │
│ ─────────────────────────── │
│ 宜：求职、表白、洽谈         │
│ 忌：投资、签约               │
│ 🕐 最佳时段：上午9-11点     │
│─────────────────────────────│
│        [查看完整报告] →       │
└─────────────────────────────┘
```

## 功能2：分阶段加载进度

### 设计方案
- **方案：** A 固定百分比（前端实现）
- **进度阶段：**
  1. 八字计算中 → 40%
  2. AI分析中 → 70%
  3. 报告生成中 → 100%

### UI表现
- 显示当前阶段名称 + 进度条 + 百分比
- 每个阶段完成后自动进入下一阶段
- 与现有 DestinyRings loading 动画结合

## 功能3：连续签到奖励池

### 设计方案
- **日常展示：** A 进度条 - "再签到3天获得补签卡"
- **详情展开：** 点击后显示里程碑卡片墙
- **达标庆祝：** 弹窗展示获得的奖励

### 里程碑设计
| 连续天数 | 奖励 |
|----------|------|
| 7天 | 补签卡×1 |
| 30天 | 高级报告解锁券×1 |

### 补签卡机制
- 用户漏签一天可用补签卡补签
- 补签后连续记录不中断

## 功能4：动态分享卡片

### 设计方案
- **方案：** C 静态升级
- **升级内容：**
  - 精美装饰边框
  - 动态色彩渐变
  - 命盘元素装饰

### 技术实现
- 使用 html-to-image 生成图片
- CSS动画实现渐变效果
- 保持现有分享流程不变
