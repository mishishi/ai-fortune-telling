## 1. 概述

### 目标
升级分享卡片设计，目标是提升口碑传播效果。

### 设计方向
**视觉冲击型** - 深色背景+金色点缀，命理神秘感，适配朋友圈/微博等社交媒体信息流场景。

### 核心价值
- 视觉冲击力强，在信息流中0.5秒内抓住注意力
- 具备社交货币属性（值得点赞/评论）
- 与产品玄学/命理主题契合

## 3. 动态效果规范

### 雷达图脉动
```css
@keyframes pulse {
  0%, 100% { opacity: 0.3; transform: scale(1); }
  50% { opacity: 0.6; transform: scale(1.05); }
}
```
- 光晕层位于雷达图下方
- 动画周期：2秒
- 渲染为静态时保留最终状态

### 流星粒子
- 3-5个流星从右上角划过
- 使用CSS动画实现
- 渲染为静态时保留轨迹

### 进度条流光
```css
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
```
- 流光效果叠加在进度条上
- 渲染为静态时保留光泽

## 6. 技术实现

### 图片生成
使用 `html-to-image` 库的 `toPng` 方法：

```typescript
const dataUrl = await toPng(cardRef.current, {
  quality: 1,
  pixelRatio: 2,  // 视网膜清晰度
  cacheBust: true,
});
```

### 动态效果处理
1. 渲染前：给动画元素添加 `.is-animating` 类触发CSS动画
2. 等待下一个动画帧（确保动画已呈现）
3. 执行截图
4. 渲染后：移除 `.is-animating` 类

### 模板选择
- 默认模板：星河命盘
- 用户可在分享弹窗中切换模板
- 切换时实时预览

## 7. 分享流程

```
用户点击「分享报告」按钮
    ↓
弹出分享弹窗（显示模板选择器）
    ↓
用户选择模板（可选）
    ↓
点击「生成分享图片」
    ↓
显示「生成中...」状态
    ↓
生成PNG图片
    ↓
尝试 Web Share API（移动端）
    ↓
不支持则自动下载
    ↓
显示成功提示
```

## 8. 验收标准

1. **三个模板都能正常渲染** - 切换模板时预览实时更新
2. **动态效果可见** - 雷达图光晕、流星动画在截图时保留最终状态
3. **水印正确显示** - 右下角品牌水印 + 左下角模板标识
4. **图片清晰度** - pixelRatio:2 确保视网膜屏清晰
5. **移动端兼容** - Web Share API 正常工作
6. **降级处理** - 不支持分享API时下载图片
