# 八字命盘环形展示设计规范

> **For agentic workers:** Implementation uses subagent-driven-development.

**Goal:** 设计并实现八字命盘的环形可视化展示，作为报告页的核心视觉焦点

**Architecture:** 在报告页首屏用环形命盘展示四柱（年/月/日/时），日柱置于中心，四柱环绕。环形代表命理轮回概念，创造独特的视觉记忆点。

---

## 1. 视觉设计

### 命盘环形布局

```
        年柱 (顶部)
          壬
          子
        
月柱(左)            月柱(右)
  丙                    辛
  寅                    酉
        
        日柱 (底部/中心)
          戊
          辰
```

### 环形结构
- 外环：四柱（年/月/日/时）均匀分布在圆周上
- 内圆：日柱（戊辰）居中，作为命盘核心
- 四柱之间用弧线连接，形成完整的"命局"感
- 每个位置显示：天干 + 地支 + 五行颜色

### 色彩系统
```css
--bazi-wood: #4ade80;      /* 木 - 绿 */
--bazi-fire: #f87171;       /* 火 - 红 */
--bazi-earth: #fbbf24;      /* 土 - 黄 */
--bazi-metal: #60a5fa;       /* 金 - 蓝 */
--bazi-water: #a78bfa;       /* 水 - 紫 */
```

### 动效设计
- 入场动画：环形从中心向外展开，staggered出现（300ms ease-out）
- Hover：悬停某柱时，该柱高亮，其他柱变暗（opacity 0.5）
- 点击：展开该柱详细信息（天干地支解读、五行生克）

## 2. 组件结构

### BaziRing 组件
```
src/components/BaziRing.tsx
```

**Props:**
```typescript
interface BaziRingProps {
  birthData: {
    yearPillar: { stem: string; branch: string; element: string };
    monthPillar: { stem: string; branch: string; element: string };
    dayPillar: { stem: string; branch: string; element: string };
    hourPillar: { stem: string; branch: string; element: string };
  };
  size?: number;  // 默认 280px
}
```

**视觉结构:**
```
┌─────────────────────────────────┐
│            年柱 壬子             │
│         (顶部，箭头向下)          │
│                                 │
│   月柱丙寅              辛酉月柱 │
│   (左侧)                  (右侧)  │
│                                 │
│            ┌───┐                │
│            │日柱│ ← 中心圆      │
│            │戊辰│               │
│            └───┘                │
│                                 │
│          时柱 壬戌              │
│         (底部，箭头向上)          │
└─────────────────────────────────┘
```

### 中心圆（日柱）
- 直径：外环直径的 40%
- 显示：日干（大字）+ 日支（小字）+ 五行颜色
- 背景：半透明渐变 + 五行颜色光晕
- 动画：轻微呼吸效果（pulse scale 1.0 → 1.02）

### 外环四柱
- 每个位置占 90度弧度
- 天干 + 地支垂直排列
- 底部显示五行颜色条
- 连接线：淡色虚线，hover时变实线高亮

## 3. 交互设计

### Hover 状态
- 悬停某柱：该柱高亮（五行颜色 + 发光），其他三柱 opacity: 0.4
- 中心圆轻微放大

### Click 状态
- 点击某柱：展开弹出卡片
- 卡片内容：该柱详细解读（天干意义、地支意义、三合/六合/冲/刑等）

### 入场动画顺序
1. 外环线条先出现（绘制动画，400ms）
2. 四柱依次出现（staggered，每柱间隔100ms）
3. 中心圆最后出现 + 呼吸动效开始

---

## 4. 位置计算

四柱按顺时针排列：
- 年柱：顶部（12点钟方向，270度）
- 月柱：左侧（9点钟方向，180度）
- 日柱：底部/中心（不是在外环，在中心圆）
- 时柱：右侧（3点钟方向，0/360度）

---

## 5. 响应式设计

- **Desktop (>768px)**: 280px 环形，居中展示
- **Mobile (<768px)**: 240px 环形，适配小屏
- 最大宽度限制：360px

## 7. 技术实现

### SVG实现
- 使用 SVG 绘制环形，确保在各分辨率下清晰
- 四柱位置用三角函数计算：angle = (index * 90 - 90) degrees
- 连接弧线用 `<path>` 绘制

### 五行颜色映射
```typescript
const ELEMENT_COLORS = {
  wood: '#4ade80',
  fire: '#f87171',
  earth: '#fbbf24',
  metal: '#60a5fa',
  water: '#a78bfa',
};
```

### 组件状态

```typescript
interface BaziRingState {
  hoveredPillar: 'year' | 'month' | 'day' | 'hour' | null;
  clickedPillar: 'year' | 'month' | 'day' | 'hour' | null;
}
```
